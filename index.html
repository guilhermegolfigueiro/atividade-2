Este trabalho tem como objetivo explicar diferentes tags presentes no html, logo a lista com explicações delas a seguir


H1 a tag permite adicionar um título, existem 6 tipos diferentes dela, h1, h2, h3...sendo h1 o maior tamanho de texto, e h6 o menor possível


br a tag permite adicionar um espaco entre doas linhas de texto, permitindo melhor organização no desenvolvimento dos textos


p a tag permite adicionar texto, ela é o que faz um parágrafo


b transforma o conteúdo em negrito


i transforma o texto em itálico
